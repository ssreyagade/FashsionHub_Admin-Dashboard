# E-Commerce Admin Dashboard

## Description
This is a **React-based Admin Dashboard** for managing an e-commerce store. It provides a clean, responsive interface for administrators to:

- Manage **products/inventory** (add, edit, delete, image upload).  
- Track and **confirm orders**.  
- Generate **invoices** for confirmed orders.  
- View dynamic **dashboard statistics** like number of employees, products, and orders.  
- Analyze sales and prices with a simple **sales analysis panel**.  

The UI is built using **Tailwind CSS** with **responsive design** for desktop and mobile. Admins can easily navigate between pages using a **sidebar menu** with toggle functionality for smaller screens.

## Features
- Dynamic **Dashboard** cards with counts fetched from JSON/API.  
- **Inventory management** with product images, colors, and reviews.  
- **Orders page** with confirmation option.  
- **Invoice page** to display confirmed orders and send invoices.  
- Fully **responsive sidebar** and **top navbar**.  

## Technologies Used
- React  
- React Router DOM  
- Tailwind CSS  
- React Icons  
- JSON Server (optional)
